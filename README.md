# KRONATRIX AI — GitHub Pages Website

This folder is ready to upload directly to the root of the GitHub repository that publishes `https://kronatrix.co.uk/`.

## Included

- `index.html` — existing KRONATRIX site with SEO, AI-search, analytics, branding, accessibility, conversion and official-network improvements
- `kronatrix-network.html` — complete directory of official KRONATRIX websites, lead forms, social profiles and marketing GPTs
- `CNAME` — custom domain configuration for `kronatrix.co.uk`
- `google10473d8cee5f5fe6.html` — Google Search Console verification
- `robots.txt` and `sitemap.xml` — crawling and indexing support
- `llms.txt` and `llms-full.txt` — machine-readable brand and official-link summaries
- `site.webmanifest`, favicons and app icons
- `404.html` — branded error page
- optimised KRONATRIX logo, wordmark, mascot, social preview and audio file
